Type 
```console
      cargo run -- argument
``` 
to pass an argument to the program.  

1. Pass an argument to the program and print that argument, for example: 
   ```console
      ./rename_files my_argument
      my_argument
   ```
1. Modify the argument by adding "-" and random alphanumeric eight characters, e.g.:  
   ```console
      ./rename_files my_argument
      my_argument-jy8o04ec
   ```
